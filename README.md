# Портфолио Жанғазы Алымұлы

Статический премиум-сайт портфолио оюшы, мастера казахского орнамента, с главной страницей, страницами работ, живым орнаментальным фоном, анимациями и переключателем языка `Қаз / Рус`.

## Файлы

- `index.html` - главная страница портфолио;
- `artwork.html?id=kartina-1` - страница первой работы;
- `artwork.html?id=kartina-2` - страница второй работы;
- `assets/js/artworks.js` - данные работ, казахские тексты и русские переводы;
- `assets/js/app.js` - переключатель языка, рендер страниц и мягкие интерактивные анимации;
- `assets/css/styles.css` - визуальная система, орнаментальный фон, адаптив и motion states;
- `assets/images/kartina-1.jpg`, `assets/images/kartina-2.jpg` - вырезанные изображения работ без текстового блока;
- `assets/images/logo.jpg` - логотип художника.

## GitHub Pages

Сайт не требует сборки. Для GitHub Pages достаточно загрузить файлы проекта в корень репозитория и включить публикацию из ветки `main` / папки root.

Служебные папки `node_modules` и `tools` публиковать не нужно.
